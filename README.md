# Users Directory App

## How to Run the Project

### Local Development
1. **Install dependencies:**
   ```sh
   npm ci
   npm run dev
   ```
   This will install dependencies (if needed) and start the Vite dev server. The app will be available at [http://localhost:5173](http://localhost:5173).
